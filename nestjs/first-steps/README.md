# first-steps

## Description

This folder will be product of following the steps i the Getting started or first-steps guide from the official documentation page.

https://docs.nestjs.com/first-steps

At the moment of this commit the versions are:

npm: 5.6.0
node: v8.11.3
nest-cli: 5.3.0

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
